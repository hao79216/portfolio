每日簽到

提升品牌活動與好友互動體驗

關於作品

題目 - 群發訊息

許多品牌會透過 LINE 官方帳號經營會員，但缺乏能夠「持續提高互動頻率」的活動工具。
為了提升用戶回訪率與品牌互動，設計了一個 LINE OA 每日簽到活動功能，讓品牌可以在指定活動期間內，鼓勵會員每天簽到並獲得不同獎勵。

產品目標:

  • 提升 LINE OA 好友的每日互動頻率
  • 鼓勵使用者持續回訪品牌官方帳號
  • 透過簽到機制建立長期互動習慣

執行時間:

2 weeks

扮演角色:

UX: 功能流程規劃
UI: Wireframe、UI Flow

設計流程

使用情境:

例如：
電商品牌在週年慶期間推出 7 日簽到活動，使用者每天進入 LINE OA 簽到即可獲得不同獎勵。
簽到完成後可以觸發 LINE Flex Message 發送獎勵或遊玩遊戲，也可以根據觸發次數去收集標籤。

簽到規則:

  • 使用者每天可簽到 1 次
  • 簽到需 連續進行，若中斷則需重新從 Day 1 開始
  • 每日簽到可獲得不同獎勵
  • 部分簽到日可觸發 額外互動（如遊戲或活動）

簽到流程:

由於每日簽到功能的使用情境相對單純，使用者主要流程為：
進入 LineOA → 簽到頁面 → 點擊簽到 → 獲得獎勵
整體操作步驟簡單且固定，因此本次設計將重點放在 活動規則設計與前後台介面規劃。

功能展示

Wirefframe

設計重點

  • 讓使用者快速理解簽到規則
  • 清楚呈現簽到進度
  • 讓後台能夠靈活設定活動與獎勵

UI 設計

在設計流程上，我先從前台使用者介面開始規劃。
透過確認使用者的簽到流程與畫面需求，整理出活動所需的規則與資料結構，再進一步設計後台設定介面，確保管理者能夠完整配置前台所需的活動內容。

設計成果

  • 完整規劃每日簽到活動機制
  • 設計前台與後台操作流程
  • 支援與 LINE OA 行銷流程整合

成效成果

此功能已在公司內部實際應用於 LINE OA 活動中，並獲得主管正面回饋。
透過每日簽到與獎勵機制，有效提升使用者回到官方帳號參與互動的意願。
由於整體活動流程與互動體驗具有良好的實用性，目前功能已正式上線，並規劃後續作為產品功能進行對外推廣。
